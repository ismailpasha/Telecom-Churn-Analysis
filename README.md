# 📊 Telecom Churn Prediction

## 🚀 Project Overview
Customer churn is a big challenge in the telecom industry. In this project, I built a machine learning model to predict which customers are likely to leave a telecom service.

The idea is simple: if we can identify customers at risk early, businesses can take action to retain them and reduce revenue loss.

---

## Project Structure
- `data/raw` → Original dataset  
- `notebooks` → Data analysis and experimentation  
- `src` → Model and preprocessing code  
- `outputs` → Results, predictions, and visualizations  

---

## Tools & Technologies
- Python  
- Pandas & NumPy  
- Scikit-learn  
- Matplotlib & Seaborn  

---

## What I Did
- Cleaned and prepared the data  
- Performed exploratory data analysis (EDA)  
- Engineered useful features  
- Trained and evaluated a machine learning model  

---

## Model Performance

- **Accuracy:** 77.1%  
- **Precision:** 55.3%  
- **Recall:** 73.0%  
- **F1 Score:** 62.9%  

## What This Means
The model does a good job identifying customers who are likely to churn (high recall), which is important from a business perspective.  

There are still some false positives (precision can be improved), but overall the model provides useful insights for retention strategies.

---

## 📈 Why This Matters
- Helps businesses identify at-risk customers  
- Supports targeted retention campaigns  
- Can reduce customer loss and improve revenue
